Slides.js

Slides.js in a nutshell…

Slides.js is a lightweight framework to generate animated slideshows using templates. Templates can be easily developed, using only CSS declarations, even for the animations and dynamic components (outline slides, background images, footer…).

Main features

Slides.js is packed with features:
  • The slides are automatically resized to adapt to the screen size.
  • By typing on the "O" key, you can get an overwiew of all slides and easily move between them.
  • Presenter notes are available and synchronized with the slideshow status, this document is an example.
  • People in the audience can get the slideshow on their own device and view it automatically synchronized with the talk.
  • Transitions between slides are animated and configurable.
  • Inside each slide, you can have several fragments which are animated individually.
  • The whole slideshow may be configured with CSS only.
  • Outlines slides are automatically generated based on headings in the source document. Outlines are fully clickable.
  • The slideshow may be browsed using the keypad or a touch device.
